# Lecture Materials

This directory contains lecture slides, notes, and supplementary materials.

## Organization

```
lectures/
├── slides/     # Presentation slides (PDF, PPTX, etc.)
├── notes/      # Lecture notes and handouts
└── code/       # Code examples and demonstrations
```

## Format Recommendations

- **Slides**: PDF format for easy sharing
- **Notes**: Markdown or PDF
- **Code**: Jupyter notebooks with clear documentation

## Sharing with Students

Materials are automatically synced to the course website via the sync script.
