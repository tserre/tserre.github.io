# Course Papers

This directory contains PDF files of papers used in the course.

## Organization

Papers should be organized by week or topic:

```
papers/
├── week01/
├── week02/
├── classics/
└── optional/
```

## Naming Convention

Use descriptive filenames:
- `AuthorYear-ShortTitle.pdf`
- Example: `Krizhevsky2012-ImageNet.pdf`

## Adding Papers

1. Add PDF file to appropriate subdirectory
2. Update `readings.yml` with paper metadata
3. Run sync script to update website

## Copyright

Ensure you have appropriate rights to distribute papers. Consider:
- Using institutional access
- Linking to open access versions when available
- Providing DOI links for paywall papers
